# 🎬 TV Show Search App

A simple and interactive web app that allows users to search for TV shows and display their posters using the TVMaze API.


# 🚀 Features

* 🔍 Search for any TV show
* 🖼️ Display show posters dynamically
* ⚠️ Handles missing images gracefully
* 🎯 Shows custom messages for selected shows
* ⏳ Loading indicator while fetching data
* ❌ Handles empty and no-result searches
* 🎨 Clean grid layout UI

---

# 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Axios
* TVMaze API

---

# 📸 Preview



## ⚙️ How It Works

1. User enters a TV show name
2. App sends a request to the TVMaze API
3. Results are fetched and displayed as images
4. Special message appears for selected shows
